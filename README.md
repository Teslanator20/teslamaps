# TeslaMaps

A Hypixel Skyblock Dungeons mod for Minecraft 1.21.10 (Fabric).

## Features

### Dungeon Map
- Full dungeon map display showing all rooms including unopened ones
- Room identification with names and secrets count
- Checkmark status tracking (white/green/failed)
- Mimic detection and ESP
- Wither/Blood door ESP and tracers

### Puzzle Solvers
- Water Board - Shows which levers to pull and when
- Tic Tac Toe - Highlights optimal move
- Three Weirdos - Identifies correct NPC
- Boulder - Shows solution path
- Creeper Beams - Highlights targets
- Simon Says - Tracks button sequence
- Quiz/Trivia - Highlights correct answer
- Blaze - Shows kill order
- Teleport Maze - Shows correct pads
- Arrow Alignment - Shows which arrows to rotate

### Terminal Solvers (F7)
- Click in Order
- Correct All Panes
- Starts With
- Select All Color
- Rubix (Change to Same Color)
- Melody

### Other Features
- Secret Waypoints - Shows secret locations in rooms
- Starred Mob ESP - Highlights minibosses
- Livid Solver - Identifies correct Livid on F5/M5
- Leap Overlay - Enhanced spirit leap menu
- Score Calculator - Live dungeon score estimation
- Auto GFS - Refills sacks on dungeon start

## Installation

1. Install Fabric for Minecraft 1.21.10
2. Install Fabric API
3. Place the mod jar in your mods folder

## Configuration

Use `/tmap` or `/tmap config` to open the settings menu.
